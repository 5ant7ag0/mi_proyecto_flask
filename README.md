🚀 Full Stack Project: Flask + PostgreSQL + Angular
🧠 Overview
This project is a practical and progressive implementation designed to learn the Flask microframework in Python, starting from the basics and advancing toward building a complete web application with a modern architecture.
Throughout the development, key technologies from today’s ecosystem are integrated, including a relational database (PostgreSQL) and a dynamic frontend built with Angular, resulting in a robust and scalable full stack solution.
🎯 Project Goal
Develop a complete web application that allows:
Understanding Flask structure and workflow from scratch
Building a functional REST API
Managing data using PostgreSQL
Connecting a Python backend with a modern Angular frontend
Applying best practices in web development
🏗️ Project Architecture
The system is divided into three main layers:
🔹 Backend (Flask)
Lightweight and flexible Python microframework
Route and controller creation
REST API implementation
HTTP request handling (GET, POST, PUT, DELETE)
Database integration
🔹 Database (PostgreSQL)
Relational database management system
Structured data modeling
Data persistence
Optimized SQL queries
🔹 Frontend (Angular)
Modern framework for web applications
Dynamic and reactive user interface
REST API consumption
Component and service architecture
🔄 Workflow
The user interacts with the Angular interface
Angular sends HTTP requests to the Flask backend
Flask processes the business logic
Database operations are executed in PostgreSQL
Flask returns the response
Angular updates the UI dynamically
